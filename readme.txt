¿Cuál es la diferencia entre un componente UI y un componente funcional en React?
El componente UI o User Interface es para la presentación, solamente se centra en como se ve la aplicación y no tiene nada de lógica y el componente funcional si contiene lógica y puede manejar su propio estado interno (useState, useEffect).

¿Qué son las props en React y cuál es su propósito principal?
Estos son una forma de pasar los datos de un componente padre a un componente hijo y su principal propósito es de flexibilidad y reutilización.

¿Qué son los children props en React y por qué no se recomienda su uso excesivo?
Los children se usan para pasar componentes hijos dentro de un componente para que las interfaces sean más modulares y flexibles y se recomienda evitar su uso excesivo por el motivo que puede generar problemas de mantenimiento y legibilidad en el código.

¿Qué es useState en React y para qué se utiliza principalmente?
useState es un hook (función especial que permite utilizar características avanzadas), que permite a los componentes funcionales tener y gestionar su propio estado.
